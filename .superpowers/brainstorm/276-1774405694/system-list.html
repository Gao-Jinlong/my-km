富文本编辑器 - 完整系统清单

基于产品需求推导的 12 个核心系统

✅ P0 核心系统（优先建设）

1. DocumentModel

文档/块的领域模型，定义 Document、Block、BlockType 等核心抽象

依赖
无 ← 这是基础层

2. BlockRegistry

块类型注册中心，管理 BlockType ↔ Lexical Node 映射

依赖
DocumentModel

3. EditorService

编辑器实例管理、状态追踪、选区/光标同步

依赖
BlockRegistry, Lexical

4. CommandService

命令注册、执行、快捷键绑定、工具栏状态同步

依赖
EditorService, BlockRegistry

5. AutoSaveService

防抖保存、版本管理、冲突解决

依赖
DocumentModel, FileSystemService

6. AIContextService

AI 上下文协议、按需请求、响应处理

依赖
EditorService

✅ P1 支持系统

7. HistoryService

撤销/重做栈管理，与自动保存协同

依赖
Lexical History

8. InputRuleService

Markdown 快捷输入规则引擎

依赖
BlockRegistry, CommandService

9. NodeViewRegistry

Lexical Node → React 组件映射

依赖
BlockRegistry

✅ P2 增强系统

10. ClipboardService

复制/粘贴/剪切，富文本、图片、跨文档拖拽

依赖
BlockRegistry, NodeViewRegistry

11. PerformanceOptimization

大文档虚拟滚动、懒加载、更新批处理

依赖
NodeViewRegistry

✅ P3 未来系统

12. CollaborationEngine

未来多人协作的 CRDT/OT 基础，操作转换

依赖
所有 P0/P1 系统
