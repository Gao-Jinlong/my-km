文档编辑器架构方案

选择一种技术架构方向

A

方案 A: Lexical 富文本编辑器

基于已安装的 @lexical/* 包，构建类似 Notion 的块编辑器，支持 Markdown 语法、表格、代码块等

优势
  • 已有依赖，无需新增
  • 强大的扩展性
  • 支持协作编辑
B

方案 B: Monaco Editor

VS Code 同款编辑器核心，强大的代码编辑能力，适合技术文档

优势
  • 业界最强的代码编辑体验
  • 完美的语法高亮
  • 支持 LSP 语言服务
C

方案 C: 混合方案

Markdown 用 Lexical，代码用 Monaco，根据文件类型自动切换

优势
  • 兼顾两者优势
  • 最专业的体验

💡 提示：点击选项可选择，可以多点选择来对比
