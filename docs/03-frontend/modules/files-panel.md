# 文件夹面板

## 功能简介

文件夹面板以树形结构展示项目文件，提供文件浏览和操作功能。

## 浏览器限制说明

> **重要**: 受浏览器安全限制，系统使用 File System Access API，文件操作能力如下：
>
> - ✅ 用户可以**选择**一个本地文件夹作为项目
> - ✅ 在**已选择的文件夹内**读取/写入文件
> - ✅ 在**已选择的文件夹内**创建新文件/文件夹
> - ❌ 无法在用户未授权的目录创建文件
> - ❌ 无法直接指定任意路径保存文件
>
> 所有"新建文件"操作都必须在**用户已打开的项目文件夹内**进行。

## 核心功能

### 文件树展示

- 递归展示文件夹层次
- 支持展开/折叠操作
- 虚拟滚动优化性能
- 仅显示 `.md` / `.mdx` 文件

### 文件操作

| 操作 | 说明 | 限制 |
|------|------|------|
| 新建文件 | 在当前选中的文件夹内创建新文件 | 必须在项目文件夹内 |
| 新建文件夹 | 在当前选中的文件夹内创建新文件夹 | 必须在项目文件夹内 |
| 重命名 | 重命名文件/文件夹 | - |
| 删除 | 删除文件/文件夹 | 删除前需确认 |
| 拖拽移动 | 在文件树内拖拽移动文件 | 不能移出项目文件夹 |

### 快速定位

- 键盘导航（上下箭头）
- 搜索过滤
- 自动滚动到当前文件

## 新建文件流程

### 场景 1: 在已有文件夹内新建

```
1. 用户右键点击目标文件夹
2. 选择"新建文件"
3. 输入文件名（默认扩展名 .md）
4. 文件创建在该文件夹内
```

### 场景 2: 在项目根目录新建

```
1. 用户右键点击文件树根节点
2. 选择"新建文件"
3. 输入文件名
4. 文件创建在项目根目录
```

### 场景 3: 通过编辑器新建（待实现）

```
1. 用户在编辑器中点击"新建文档"
2. 系统弹出"选择保存位置"对话框
3. 用户在项目文件夹内选择目标目录
4. 输入文件名并保存
```

## 右键菜单

| 操作 | 说明 | 可用条件 |
|------|------|----------|
| 新建文件 | 在当前文件夹内创建新文件 | - |
| 新建文件夹 | 在当前文件夹内创建新文件夹 | - |
| 重命名 | 重命名文件/文件夹 | 选中单个项目 |
| 删除 | 删除文件/文件夹 | 选中一个或多个项目 |
| 在文件系统中显示 | 打开系统文件管理器定位 | 支持 File System Access API 的浏览器 |
