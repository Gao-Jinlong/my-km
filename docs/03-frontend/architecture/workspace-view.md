# 工作视图模块

## 概述

工作视图是应用的核心工作区，提供类 VSCode 风格的三栏布局。

## 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  Top Nav: Logo | 项目名称 | [搜索] [AI] [侧边栏]            │
├──────┬────────────────────┬─────────────────────────────────┤
│      │                    │  Tab1  Tab2  Tab3               │
│ 侧边 │   Sidebar Panel    │ ┌─────────────────────────────┐ │
│ 栏   │   (Files/Search)   │ │      Editor Content         │ │
│      │                    │ └─────────────────────────────┘ │
├──────┴────────────────────┴─────────────────────────────────┤
│  Status Bar: Ln 1, Col 1 | UTF-8 | Markdown                │
└─────────────────────────────────────────────────────────────┘
```

## 核心组件

| 组件 | 说明 |
|------|------|
| 顶部导航栏 | Logo、项目名称、全局操作 |
| 左侧侧边栏 | 文件树、搜索面板 |
| 编辑器区域 | Tab 栏、编辑内容区 |
| AI 面板 | 对话历史、输入框 |
| 底部状态栏 | 光标位置、编码、语言 |

## 依赖模块

- **文件系统**: 文件树和文件操作
- **编辑器**: Markdown 编辑
- **AI 对话**: 智能助手
- **项目管理**: 项目信息
