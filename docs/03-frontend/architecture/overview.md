# Sidebar 模块概述

## 功能简介

Sidebar 模块提供工作视图左侧面板功能，包括：

- **Tab 切换系统**: 动态添加/删除面板，拖拽调整顺序
- **文件夹面板**: 文件树展示和文件操作
- **搜索面板**: 全局搜索功能

## 布局结构

```
┌─────────────────────────┐
│  [Files] [Search] [+]  │ ← Tab 切换栏
├─────────────────────────┤
│   子面板内容区域         │
├─────────────────────────┤
│  [⚙️]  [👤]            │ ← 底部操作区
└─────────────────────────┘
```
