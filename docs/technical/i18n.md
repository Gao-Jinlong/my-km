## 📝 添加翻译

### 1. 编辑翻译文件

**中文**: [apps/web/messages/zh-CN.json](../../apps/web/messages/zh-CN.json)
**英文**: [apps/web/messages/en.json](../../apps/web/messages/en.json)

```json
{
  "yourFeature": {
    "title": "功能标题",
    "description": "功能描述"
  }
}
```

### 2. 在组件中使用

**客户端组件**:
```tsx
'use client';
import { useTranslations } from 'next-intl';

export function YourComponent() {
    const t = useTranslations('yourFeature');
    return <div>{t('title')}</div>;
}
```

**服务端组件**:
```tsx
import { getTranslations } from 'next-intl/server';

export default async function ServerComponent() {
    const t = await getTranslations('yourFeature');
    return <div>{t('title')}</div>;
}
```

### 3. 参数化翻译

```json
{
  "welcome": "欢迎, {{name}}!",
  "itemCount": "你有 {{count}} 个项目"
}
```

```tsx
t('welcome', { name: 'John' })
t('itemCount', { count: 5 })
```

## 🔧 后端使用

### API 接收语言

前端自动发送 `X-Locale` header，后端使用装饰器接收:

```typescript
import { CurrentLocale } from '@/i18n';

@Controller('auth')
export class AuthController {
    @Post('register')
    async register(
        @Body() dto: RegisterDto,
        @CurrentLocale() locale: string, // 自动注入: 'zh-CN' 或 'en'
    ) {
        return this.authService.register(dto, locale);
    }
}
```

### 翻译错误消息

```typescript
import { I18nService } from '@/i18n';

@Injectable()
export class YourService {
    constructor(private readonly i18nService: I18nService) {}

    yourMethod(locale: string) {
        const message = this.i18nService.getErrorMessage(
            'ERROR_CODE',
            locale
        );
        return { message };
    }
}
```

### 添加新的错误消息

编辑 [apps/server/src/i18n/constants/error-messages.ts](../../apps/server/src/i18n/constants/error-messages.ts):

```typescript
const errorMessages: Record<Locale, Record<ErrorCode, string>> = {
    'zh-CN': {
        YOUR_NEW_ERROR: '新的中文错误消息',
    },
    en: {
        YOUR_NEW_ERROR: 'New English error message',
    },
}
```

## 🌐 常见任务

### 创建翻译链接

```tsx
import { Link } from '@/i18n/routing';

<Link href="/projects">项目</Link>
<Link href="/about">关于</Link>
```

### 编程式语言切换

```tsx
'use client';
import { useRouter, usePathname } from '@/i18n/routing';

export function LanguageSwitcher() {
    const router = useRouter();
    const pathname = usePathname();

    const switchToEnglish = () => {
        router.replace(pathname, { locale: 'en' });
    };

    return <button onClick={switchToEnglish}>English</button>;
}
```

### 获取当前语言

```tsx
'use client';
import { useLocale } from 'next-intl';

export function CurrentLocaleDisplay() {
    const locale = useLocale(); // 'zh-CN' 或 'en'
    return <div>当前语言: {locale}</div>;
}
```

### 添加新语言

1. 创建新的翻译文件 `/apps/web/messages/{locale}.json`
2. 更新 [apps/web/src/i18n/config.ts](../../apps/web/src/i18n/config.ts):

```typescript
export const locales: Locale[] = ['zh-CN', 'en', 'ja']; // 添加日语

export const localeNames: Record<Locale, string> = {
    'zh-CN': '简体中文',
    'en': 'English',
    'ja': '日本語',
};
```

3. 更新 [apps/web/src/i18n/routing.ts](../../apps/web/src/i18n/routing.ts):

```typescript
export const routing = defineRouting({
    locales: ['zh-CN', 'en', 'ja'],
    defaultLocale: 'zh-CN',
});
```

## 🎨 最佳实践

### 翻译文件组织

使用命名空间保持结构清晰:

```json
{
  "auth": {
    "login": { "title": "..." },
    "register": { "title": "..." }
  },
  "dashboard": { "title": "..." },
  "validation": { "email": "..." },
  "errors": { "serverError": "..." }
}
```

### 命名规范

- **翻译 key**: 使用 camelCase (例如: `forgotPassword`)
- **命名空间**: 使用功能模块名 (例如: `auth.login`)
- **保持一致性**: 相同概念使用相同 key

### 参数化翻译

**推荐**:
```json
{ "welcome": "欢迎, {{name}}!" }
```

**不推荐**:
```json
{ "welcome": "欢迎!" }
{ "welcomeJohn": "欢迎, John!" }
```

## 🐛 常见问题

### Q: 翻译不显示怎么办？

1. 检查翻译文件 JSON 格式是否正确
2. 确认 namespace 和 key 名称正确
3. 查看浏览器控制台是否有错误

### Q: 语言切换后 URL 没变？

确保使用 `Link` from `@/i18n/routing`:

```tsx
import { Link } from '@/i18n/routing'; // ✅ 正确
import Link from 'next/link';           // ❌ 错误
```

### Q: 构建失败？

```bash
# 清理缓存
rm -rf .next
pnpm build
```

### Q: 后端收到的 locale 是 undefined？

1. 检查 API 客户端是否正确发送 `X-Locale` header
2. 确认 I18nMiddleware 已在 main.ts 中注册
3. 验证 @CurrentLocale() 装饰器使用正确
