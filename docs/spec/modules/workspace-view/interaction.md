## 🎯 模块概述

键盘快捷键、面板折叠展开等交互功能。

---

## 📖 功能需求

### WV-FR-10: 键盘快捷键

**优先级**: SHOULD

**描述**:
提供丰富的键盘快捷键,提升操作效率。

**验收标准**:
- [ ] `Cmd/Ctrl + B`: 切换左侧侧边栏
- [ ] `Cmd/Ctrl + J`: 切换右侧 AI 面板
- [ ] `Cmd/Ctrl + \`: 拆分编辑器
- [ ] `Cmd/Ctrl + 1/2/3/4`: 切换到第 1/2/3/4 个编辑器窗口
- [ ] `Cmd/Ctrl + Tab`: 切换到下一个 tab
- [ ] `Cmd/Ctrl + Shift + Tab`: 切换到上一个 tab
- [ ] `Cmd/Ctrl + W`: 关闭当前 tab

**实施状态**: ⏳ 待实现

---

### WV-FR-11: 面板折叠和展开

**优先级**: MUST

**描述**:
所有面板支持折叠和展开,用户可以根据需要调整工作区布局。

**验收标准**:
- [ ] 左侧侧边栏顶部有折叠按钮
- [ ] 右侧 AI 面板顶部有折叠按钮
- [ ] 折叠后面板宽度变为 48px,显示图标提示
- [ ] 鼠标悬停在折叠面板上时显示完整内容
- [ ] 再次点击折叠按钮展开面板
- [ ] 折叠状态持久化到 LocalStorage
- [ ] 折叠/展开动画流畅

**实施状态**: ⏳ 待实现
