# Sidebar 模块

## 📋 模块简介

Sidebar 模块是工作视图的核心组件之一,提供灵活的 Tab 切换系统、动态面板管理和高效的文件浏览功能。

### 核心特性

- **灵活的 Tab 系统**: 支持动态添加/删除面板,拖拽调整顺序
- **动态面板架构**: Service + View 分离设计,支持面板动态挂载/卸载
- **高效的文件浏览**: 直观的文件树展示,快速定位文件
- **强大的搜索能力**: 多种搜索方式,快速找到所需内容
- **状态持久化**: 记住用户的自定义配置和状态
