# 产品规划

## 🎯 产品愿景

打造一个 **AI 原生的个人知识工作室**，从工作流层面重新思考知识管理，让 AI 深度融入知识生产的每个环节：收集、整理、写作、发布。

**核心价值主张**:
- 项目化管理：系统化组织知识，而非碎片化笔记
- AI 深度对话：从零开始，AI 帮助搭建知识框架
- 智能引用网络：构建知识图谱，发现隐秘关联
- 知识分享平台：将知识沉淀转化为职业价值

**目标用户**: 技术写作者、独立开发者、研究员等长期积累和知识管理的个人用户

## 🔮 技术演进

### 当前技术栈
- **前端**: Next.js 14, shadcn/ui, Tailwind CSS
- **后端**: NestJS 10, Prisma
- **数据库**: PostgreSQL + pgvector
- **AI**: 智谱AI, 阿里云百炼

- **前端**: Server Components 优化
- **后端**: 微服务拆分（如果需要）
