# 产品规划

## 🎯 产品愿景

打造一个善于归纳、检索、和持续迭代的个人知识库，用于发布、分享个人知识，构建个人的职业价值展示平台

---

## 📅 版本规划

### v0.1 - 系统基建
- [x] 技术选型
- [ ] 用户、身份验证系统
- [ ] 容器化

### v1.0 - MVP（最小可行产品）

**目标**: 实现核心的知识管理功能

**功能范围**:
- ✅ 文章 CRUD（创建、读取、更新、删除）
- ✅ Markdown 编辑和预览
- ✅ 分类和标签管理
- ✅ 关键词搜索
- ✅ 基础 UI 和布局
- 站点发布

**预计完成**: 2026-01-31

---

### v1.1 - 智能增强

**目标**: 引入 AI 能力，提升搜索和编辑体验

**新增功能**:
- 🔍 语义搜索（基于 Embedding）
- 🔍 混合搜索（关键词 + 语义）
- 🤖 AI 问答（RAG）
- 🤖 内容润色
- 🤖 生成摘要
- 🤖 提取关键词

**预计完成**: 2026-02-28

## 🔮 技术演进

### 当前技术栈
- **前端**: Next.js 14, shadcn/ui, Tailwind CSS
- **后端**: NestJS 10, Prisma
- **数据库**: PostgreSQL + pgvector
- **AI**: 智谱AI, 阿里云百炼

### 潜在技术升级
- **前端**: Server Components 优化
- **后端**: 微服务拆分（如果需要）
- **搜索**: 专用向量数据库（如 Milvus）
- **AI**: 多模型支持、Fine-tuning
